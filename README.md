# UnityVault

## Project info

UnityVault is a Vite + React + TypeScript app using shadcn-ui and Tailwind CSS.

## Run locally

1. Install dependencies:

```sh
npm install
```

2. Start the dev server:

```sh
npm run dev
```

3. Build for production:

```sh
npm run build
```

4. Preview the production build:

```sh
npm run preview
```

## Tech stack

- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS
